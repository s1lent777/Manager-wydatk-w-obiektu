Manager Wydatków Obiektu (Hotel Expense Manager)

Status projektu: Komercyjne wdrożenie + Projekt Inżynierski (Lubelska Akademia WSEI)

📋 O projekcie

Aplikacja została pierwotnie opracowana i wdrożona jako dedykowane rozwiązanie dla działającego obiektu hotelarskiego w celu optymalizacji procesów kontrolingu kosztów. Ze względu na swoją innowacyjność i skuteczność, system został rozwinięty oraz opisany jako praca inżynierska na kierunku Informatyka (specjalność: Inżynieria Oprogramowania i Technologie Mobilne).

Głównym zadaniem aplikacji jest monitorowanie wskaźników Food Cost w czasie rzeczywistym oraz automatyzacja ewidencji zakupów i obłożenia obiektu.

✨ Kluczowe funkcje

Analiza Food Cost: Automatyczne wyliczanie kosztu "wsadu do kotła" na jednego gościa.

Zarządzanie operacyjne: Rejestracja liczby śniadań, pokoi oraz gości.

Dynamiczne formularze: Możliwość wprowadzania faktur zbiorczych z wieloma pozycjami jednocześnie.

Analityka (BI): Wizualizacja struktury kosztów według kategorii (nabiał, mięso, chemia itp.) za pomocą interaktywnych wykresów.

Architektura Local-First: Pełna funkcjonalność offline, dane przechowywane bezpiecznie w przeglądarce użytkownika (LocalStorage).

Eksport danych: Generowanie raportów PDF oraz kopii zapasowych w formacie JSON.

🛠️ Technologie

Frontend: JavaScript (ES6+), HTML5, Tailwind CSS.

Wizualizacja danych: Chart.js.

Ikony: Lucide Icons / Font Awesome.

Narzędzia: Visual Studio Code, Git.
